## 🎨 Примеры комбинаций

### 🖼️ Классический портрет
```
Ввод: "Молодая женщина с книгой у окна"

Выбрать:
✓ Стиль: Photographic
✓ Техники: 
  - Oil Painting (Масляная живопись)
  - Chiaroscuro Painting (Кьяроскуро)
✓ Освещение: golden hour lighting

Результат: Реалистичный портрет в стиле старых мастеров
```

### 📚 Книжная иллюстрация
```
Ввод: "Кот в волшебном саду с бабочками"

Выбрать:
✓ Стиль: Fantasy Art
✓ Техники:
  - Pen and Watercolor Painting
  - whimsical (причудливый)
✓ Цвета: pastel colors

Результат: Нежная акварельная иллюстрация для детской книги
```

### 🎮 Игровая графика
```
Ввод: "Рыцарь сражается с драконом"

Выбрать:
✓ Стиль: Game Retro Arcade
✓ Техники:
  - Pixel Art
✓ Детализация: 8-bit graphics

Результат: Ретро-спрайт для 8-битной игры
```

### 🎪 Винтажный постер
```
Ввод: "Музыкальный фестиваль в парке"

Выбрать:
✓ Стиль: Artstyle Pop Art
✓ Техники:
  - Risograph Print
✓ Цвета: vibrant colors

Результат: Яркий постер в стиле 80-х
```

### 🎭 Экспрессивное искусство
```
Ввод: "Закат над морем"

Выбрать:
✓ Стиль: Artstyle Impressionist
✓ Техники:
  - Impasto Painting

Результат: Объёмная картина с выразительными мазками
```

### 🎯 Логотип
